## Step 1 — Install the extension

Install SyncTabs in every Chromium browser where you want to see tabs.

**From browser stores** *(auto-updates)*:

| Browser | Store |
|---------|-------|
| Google Chrome | Chrome Web Store *(coming soon)* |
| Microsoft Edge | Edge Add-ons *(coming soon)* |
| Brave / Vivaldi / Opera | Use the Chrome Web Store link |

**Or sideload manually**:

1. Download the repo and unzip
2. Open `chrome://extensions` in your browser
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked** and select the `extension/` folder
5. Pin the SyncTabs icon to your toolbar

Repeat in every browser you want to sync.

## Step 2 — Install the companion (for cross-browser sync)

The companion runs a local WebSocket server on `127.0.0.1:9234`. It never connects to the internet.

Download from [GitHub Releases](https://github.com/harshvasudeva/sync-it-up/releases/latest):

<Tabs>
  <Tab title="Windows">
    1. Download `synctabs-companion-windows-amd64.exe`
    2. Double-click to run — a tray icon appears
    3. Right-click the tray icon → **Start with Windows** to auto-start on login

    > Windows may show a SmartScreen warning (unsigned binary). Click **More info → Run anyway**.
  </Tab>
  <Tab title="macOS">
    ```bash
    chmod +x synctabs-companion-darwin-arm64
    xattr -d com.apple.quarantine synctabs-companion-darwin-arm64
    ./synctabs-companion-darwin-arm64
    ```

    A tray icon appears. Right-click → **Start with Login** to auto-start.
  </Tab>
  <Tab title="Linux">
    ```bash
    # Option A: .deb package (Ubuntu/Debian)
    sudo dpkg -i synctabs-companion_<version>_amd64.deb
    synctabs-companion &

    # Option B: Binary
    chmod +x synctabs-companion-linux-amd64
    ./synctabs-companion-linux-amd64 &
    ```
  </Tab>
</Tabs>

## Step 3 — Grant localhost permission

The extension needs permission to talk to `127.0.0.1`:

1. Click the SyncTabs toolbar icon → open popup
2. Click **Settings** (gear icon)
3. Click **Grant localhost permission**
4. Confirm in the browser prompt

The extension auto-detects the companion within a few seconds. The status dot turns **green**.
